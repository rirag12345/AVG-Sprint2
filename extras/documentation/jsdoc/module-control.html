Module: control

Control module for managing smart devices in a home. Managing heating in rooms by commanding thermostats based on temperature values from temperature sensors. Using treshold to decide if heating should turned up or down. Also creates a log file for temperature values. Using integration technique message exchange via MQTT. Control acts as consumer and producer for messages.
